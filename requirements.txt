# PCPCS 依賴
#
# 無需安裝任何額外套件！
# PCPCS 只使用 Python 標準庫。
#
# 需要的標準庫:
# - tkinter (GUI) - Linux 可能需要: sudo apt install python3-tk
# - socket (網路)
# - threading (多執行緒)
# - json (資料序列化)
# - subprocess (Ping 測試)
# - os, sys, platform (系統操作)
#
# Python 版本需求: >= 3.7
#
# ============================================
# 運行方式 (How to Run)
# ============================================
#
# 直接運行:
#   python main.py
#
# ============================================
# 打包說明 (Build Instructions)
# ============================================
#
# 如需打包成獨立可執行檔:
#   1. pip install pyinstaller
#   2. python build.py
#
# 打包後的檔案在 dist/ 目錄
# 用戶無需安裝 Python 即可運行
